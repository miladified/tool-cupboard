#### Show all of the status from sub module
`git submodule foreach git status`

#### To update and init the submodule
`git submodule update --init external-tool-cupboard/BurpSuite-Config`
#### Update All
`git submodule update --init --recursive`

#### To add new submodule:
`git submodule add https://github.com/example/library.git external/library`
